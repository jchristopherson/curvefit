Computes the median of a data set.

Parameters
[in,out]xThe data set whose median is to be found. Ideally, the data set should be monotonically increasing; however, if it is not, it may be sorted by the routine, dependent upon the value of srt. On output, the array contents are unchanged; however, they may be sorted into ascending order (dependent upon the value of srt).
[in]srtAn optional flag determining if x should be sorted. The default is to sort (true).
Returns
The median of x.
