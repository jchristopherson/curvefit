Computes the z-value (z-score) given a percentage of the area under the standard normal distribution curve.

Parameters
[in]alphaThe percentage of the area under the curve. This value must be between 0 and 1 such that: 0 < alpha < 1.
[out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • CF_INVALID_INPUT_ERROR: Occurs if alpha is does not satisfy: 0 < alpha < 1.
Returns
The z-score or z-value by solving for z where: alpha = ERF(z / sqrt(2)), where ERF is the error function.
